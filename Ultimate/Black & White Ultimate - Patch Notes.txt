v1.00	- Initial Release!

v1.01   - Removed Crocodile from the rock throwing quest. This was a bug.
        - (Hard Mode) Fixed issue with AI where Creature would constantly attack a town.
        - (Hard Mode) Adjusted AI to not make decisions as fast. The previous speed resulted in permanent creature attacks.
        - (Hard Mode) Nerfed Lethys's Lightning Bolt on Land 2 (around 50%).
		
v1.02   - (Hard Mode) Adjusted AI when freezing/itchy a Creature. The creature will now have a few seconds to act once the freeze/itch has worn off.
        - (Hard Mode) Fixed issue where AI Creatures could constantly break out of Creature fights when attacking a town.
        - (Hard Mode) Fixed issue where Nemesis would instantly capture towns on Land 5.
        - (Hard Mode) Fixed issue that reloading the game would re-enable virtual influence (hand outside of influence).
        - (Hard Mode) Reduced player's influence to 25% (from 33%)
        - (Hard Mode) Reduced the rate of belief to 75% (was 50%). Impressing towns should be a bit easier now. Belief amount/caps remain unchanged.
        - (Hard Mode) Players can now create artifacts. Slower than Normal Mode, but now possible.
        - (Hard Mode) Added event in Land 3 that results in a Game Over if Lethys's influence reaches the player's temple.

v1.03   - Pathing has been "fixed". Villagers will now ignore altitude, meaning they won't try run around (wallhug) tiny hills and get stuck forever etc. However, villagers will now also ignore water (causing them to walk over water), but it's rare enough villagers are in a position to do this. Definitely much better!
        - Custom appearances will now be active during Skirmish Mode. (Custom appreances are only disabled during online Multiplayer Games).
        - Skirmish AI colours/temples will now correctly load.
        - Replaced text references such as "Black & White: Creature Isle" to "Black & White: Ultimate".
        - Beating Land 5 should no longer cause the game to be stuck in day time permanently anymore.
        - Khazar now gives the workshop at beginning of Land 2, rather than after building the worship site. 
        - (Hard Mode) Pause is no longer disabled during the stats screen (ESC) or while in the temple. Only manually Pausing the game is disabled now (pressing P). 
        - (Hard Mode) AI speed increased.
        - (Hard Mode) AI will now create more disciples, making their towns much more efficient/larger etc.
        - (Hard Mode) AI shouldn't place weird/random buildings anymore.
        - (Hard Mode) AI Creature actions (Capture, Attack etc.) cooldown has been reduced.
        - (Hard Mode) Fixed issue where Lethys would permanently have the incorrect Symbol on Land 3.
        - (Hard Mode) Fixed issue where Nemesis would permanently have the incorrect Temple on Land 5.
        - (Hard Mode) Belief caps have been removed. To balance this, belief rates have been reduced (75% -> 50%).
        - (Hard Mode) AI Creatures will now have an easier time impressing towns.
        - (Hard Mode) Lethys Creature won't leave after Khazar is killed but will now leave when the player's creature is captured by Lethys.
        - (Hard Mode) Added event in Land 2 that results in a Game Over if Lethys's influence reaches the player's temple.
        - (Hard Mode) All towns will now only require 1,000 belief to capture.
        - (Hard Mode) Increased player's influence back to 33% (from 25%). This is due to the AI upgrade.
        - (Hard Mode) Slightly reduced the town influence rate for Nemesis on Land 5 (60% -> 50%).
        - (Hard Mode) Increased the temple influence in Land 5 (100% -> 250%). The norse workshop will no longer be out of reach now.

v1.04   - Fixed bug where Lethys could sometimes be inactive in Land 3.
        - Fixed issue where AI would occasionally have the incorrect symbol when entering a new land.
        - Vortex to Land 3 camera should be much smoother.

--------- Big Multiplayer Update! ---------
v1.10	- Multiplayer Appearance Customisation (Each player can now select their own Colour & Temple to use online. (Note: Players can have the same colours))
        - Multiplayer Saving (Multiplayer games will now automatically save every 30 seconds, meaning Online Games can be resumed at a later point. The game will save the map file + .lnd (landscape) file as "oos.txt" + "oos.lnd". (Note: Occasionally, players may see "PLAYER could not download the map" when attempting to resume an Online Game. This is due to slightly different "oos.txt" files. Simply send your own "oos.txt" to the other Player(s) to resolve the issue)).
        - Multiplayer Teams (Players can team up by leashing their Creature's to each other's temples with the compassion leash (like Khazar behaviour in Land 2). (Note: Doing the same with the aggression leash will make allies enemies again)).
        - All Creatures now have their stats set to 0 when an Online Game starts.

--------- Creature Isle Update! ---------
v1.20	- Creature Isle is now playable in Ultimate! The original game will now be known as "Classic". You can save and load games freely between Classic and Creature Isle!

v1.21	- Huge rework on custom code used for Ultimate. The game should no longer crash from random memory exceptions (just typical Black & White crashes now lol).
        - Nemesis curse no longer affects the Creature's height (temporary change while bugs are addressed).
        - Fixed an issue that made the Creature Isle building trial impossible to complete.
        - Now supports resolutions above 1920 x 1080 (1080p).
        - Selecting temple at Intro Menu is now much faster.

--------- QoL Update! ---------
v1.30   - Tyke can now be used in the Classic story. Players can now click on a small Egg beside their temple to toggle Tyke on/off.
        - Infinite Draw Distance patch has been added as an optional setting. It can be toggled on/off at the Intro menu or by pressing the following keys at any time:
            - "-" (toggle off).
            - "+" (toggle on).
        - Land 5 Creature Curse has been reworked. Instead, the Creature is now affected by Creature spells for a duration. The duration increases each time the Creature is cursed (starting at 1 min and increasing in increments of 15 seconds, up to 3 mins).
        - Villagers longer walk across water.
        - The "Skip straight to Creature select" and "Skip all of Creature tutorial" options have been restored.
        - The Ogre fight on Land 1 is now available when skipping Land 1. 
        - The "Creature Stats" tab now displays Tyke more neatly.
        - Fixed bug where capturing a town could occasionally crash the game.
        - (Hard Mode) Hard AI will now attack towns (if in their own influence).
        - (Hard Mode) Hard AI movement speed has been reduced (still much faster than default AI).
        - (Hard Mode) Hard AI will prioritise capturing towns slightly more than attacking other Gods. If no towns are available to capture, they will attack more.
        - (Hard Mode) Hard AI will no longer attack Creatures outside of their influence (Bug Fix).
        - (Hard Mode) Hard AI will no longer perform actions while cutscenes are active.
        - (Hard Mode) Khazar no longer has Hard Mode AI.
        - (Hard Mode) Pressing "P" in Hard Mode will now open the "Esc" main menu instead of doing nothing.
        - (Hard Mode) Starting/temple influence has been increased in all Lands.
        - (Hard Mode) Lethys starting population reduced by about 25-30%.
        - (Hard Mode) Belief to capture towns have been normalised. Capturing towns should be a bit easier in Hard Mode now. The beliefs are now the following:
            - Land 2: 500
            - Land 3: 500
            - Land 4: 750
            - Land 5: 1000
        - (Hard Mode) "SET_GLOBAL_LAND_BALANCE" (Impressiveness, Wood from trees, Artifact Speed etc.) have been normalised across all Lands.
        - (Hard Mode) Tyke/Apprentice reward now gives warning if Player has an existing Tyke/Apprentice.

v1.31   - Alignment music will now properly play (Bug Fix).
        - Infinite Draw Distance patch is now marked as "Unstable" in the Intro menu as it can cause FPS drops and occasionally game crashes (use at own risk!).
        - (Hard Mode) Nemesis will now start with a small amount of belief in each neutral town on Land 5.
        - (Hard Mode) Lethys will now ignore Khazar's Creature on Land 2.
        - (Hard Mode) Hard Mode AI will now properly attack Creatures in their influence.

v1.32   - Players can now select their starting Tribe in the Classic & Creature Isle story modes.
        - Creatures hold capacity has been greatly increased and normalised (difference between all Creatures is much lower). Additionally, the Crocodile hold limit bug has been fixed.
        - Intro Menu animation improvements and now displays the Player's selected Tribe & God symbol.
        - God symbols now always load correctly for AI (Before, Khazar could occasionally have Lethys symbol and vice versa etc.)
        - Infinite Draw Distance patch should no longer cause large FPS drops after a few minutes.
        - The 3rd Missionary (with accordian) now always his accordian on Land 5 and the Creature Isle intro.
        - (Bug Fix/Hard Mode) Fixed issue where typing "P" during Hard Mode would cause weird things to happen (such as in Game Save room).
        - (Bug Fix) Fixed fatal bug related to Villager pathing that could crash the Player's game.
        - (Bug Fix) Fixed bug where Khazar would occasionally not give a workshop to the Player.
        - (Creature Isle) No longer forces the Creature to be over a certain height (50% of max) or grant the basic starting spells (Food, Wood, Water).
        - (Creature Isle) Tyke's egg size has been reduced.
        - (Creature Isle) Gorilla's "Smashing" trial adjustments. It's now easier to "smash" the buildings and "smashing" the buildings is now way more responsive.

v1.33   - Added Randomiser mode for Classic & Creature Isle, that has two options:
            - (Tribes) All Tribes are random.
            - (Tribes & Spells) All Tribes and Spells are random.
        (Additionally, Players can hold 'R' while loading a Skirmish to enable the "Tribes & Spells" randomiser. This is temporary until a proper settings option is added).
        - (Bug Fix) Fixed crash that could occur when map population got very large.

--------- The Ogre Update! ---------
v1.40 	- Ogres are now playable Creatures!
			- Ogres can now cast spells.
			- Ogres can now be interacted with.
			- Ogres now have proper alignment skins.
			- Extra animations slots have been added (existing animations).
			- Crashes related to Ogres have been fixed.
		- Creature Variants/Forms have been added (Only Ogre has variants but more will be added in future):
			- 4 variants for Ogre.
				- Creature Isle form.
				- (New) Original Black & White form.
				- (New) Alien/Humanoid form.
				- (New) Gremlin form.
		- New quest "The Sad Ogre" has been added on Land 4. Look after the banished sad Ogre for 30 minutes to unlock the Ogres as playable Creatures.
		- The Land 1 Ogre fight is now accessible when selecting "Skip Land 1" option.
		- Sleg on Land 4 is now evil (red), making him stand out from the other Ogres.
		
		- The old hacky custom intro menu is gone and has been completely reworked (long overdue). It also has a new features have been added.:
			- It is now a proper menu with buttons, sliders etc.
			- It is now separated into 3 tabs:
				- "Play Game" - The settings to use when starting a fresh game. Not accessible once the game has started.
				- "Customisation" - Change any appearance related settings here, such as Colour, Temple & Creature (Creature Isle Brotherhood Ring etc.). Accessible at any time.
				- "Settings - QoL features and tweaks, such as the "Unlimited Draw Distance" patch. Accessible at any time.
			- The menu can be opened/closed during gameplay by pressing "+".
			- Added a new "Reset All" button that resets all intro menu values back to their default values.
			- The "Creature Isle Brotherhood Ring" can now be toggled on/off at any time.
			- "Eternal Special Villagers" option added!
				- When toggled on, Villagers with unique skins (Sable, Hermit etc.) will no longer die from old age, allowing them to live forever. They can still die from other causes.
			- "Creature Building" option added!
				- Toggling this off prevents Creatures from building or creating new buildings, just like the original Black & White. They may still sometimes attempt to but no building will be created/built.
			- Tribe & Spell randomisers can now be selected independently of each other. Before, if you wanted to randomise Spells, you had to randomise Tribes too.
			- Selected Tribes now take priority over the randomiser, meaning you can select a Tribe but randomise all other Tribes.
			- You can now select a Tribe when the Tribe randomiser is enabled.
			- Custom appearances (Colour & Temple) no longer reset when starting a new game. They now persist and can be reset to default values with the new "Reset All" button.
			- Custom appearances (Colour & Temple) can now be changed at any time. Before, this was only accessible when a new game was started.
			- Only settings under the "Play Game" tab are speific to that save (randomiser, tribe, difficulty etc.). Settings under "Appearance" and "Settings" tabs will persist over all saves.
			- This new menu is not accessible during Multiplayer.
		
		- Creature files are now periodically backed up, up to 100 backups. These backups can be found at "Ultimate\Scripts\CreatureMind\<CreatureFilename>.erc". If your game/Creature corrupts, you can restore them from here!
		- Ultimate now uses "ddraw_compat.dll", increasing compatibility across all machines, especially MAC/Linux.
		- Misc code clean ups and improvements, game should run a bit faster.
		- Multiplayer colours are now visible in the lobby. 
		- (Vanilla) End credits sequence now ends after a few seconds instead of waiting for the user to press the Space key.
		- (Bug Fix) Crocodile & Chicken tooltips now display the correct text.
		- (Bug Fix) Polar Bear village totem has been corrected. The original developers accidentally set it to the Ape.
		- (Bug Fix) Creature Curse height/alignment bug have finally been fully fixed.
		- (Bug Fix) Fixed bug where player couldn't activate Land 2 Worship Site quest if the Player used the randomiser.
		- (Bug Fix) Fixed crash related to maps being too large.
		- (Bug Fix) If a game is unable to save, it should give an error message more often instead of just crashing.
		- (Bug Fix) Swapping your Creature no longer reduce its height. Swapping used to reduce height very slightly due to the number rounding bug.
		- (Bug Fix) Corrected camera path issues for a few quests:
			- Intro Menu (Sharks & Father pathing) and slight improvements to the scene.
			- Missionary Land 1 (Verse 2 with camera pan to boat).
			- Choose your Creature (from gate -> Creatures).
			- Lost Brother.
			- Singing Stones Circle (completion camera).
